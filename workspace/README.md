There are node.js examples in this project.

- backend -
	[basic javascript's variable usage]
	./node.exe SourceCode/backend/basic/variable.js

	[basic javascript's for and function usage]
	./node.exe SourceCode/backend/basic/for_function.js

	[hello world]
	./node.exe SourceCode/backend/helloworld/helloworld.js

	[爬蟲餐廳]
	./node.exe SourceCode/backend/crawler/crawler_restaurant.js

	[爬蟲天氣](需輸入中央氣象局 API Key)
        ./node.exe SourceCode/backend/crawler/crawler_weather.js

	[Telegram Bot 回覆訊息](需輸入 BotFather 給的 Bot Token)
	./node.exe SourceCode/backend/bot_reply/bot_normalmessage.js

- complete_version -
	[完整範例] (需輸入 BotFather 給的 Bot Token) 和 (需輸入中央氣象局 API Key)
	./node.exe SourceCode/complete_version/csstudiotestbot.js
